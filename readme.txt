Генератор форм, предназначенный для работы с тогровой платформой TradeSmarter


== Description ==

Плагин предназначен для создания форм регистрации и авторизации, которые передают данные в торговую платформу TradeSmart. Формы создаются с помощью шорткодов.

Пример:

- Короткая форма регистрации:

[vospari_short url = "http://example.com/test" width = "350"]

, где атрибуты: url - перенаправление на любой url (обязательно), width - ширина формы (если не указана то auto), a_aid - партнерский id (необязательно, если указан, то будет "зашит" в коде), policy - условия и положения (необязательно, принимает 1 или 0), value - надпись на кнопке (необязательно, если не указан то "Отправить"), practice - демосчет(1) или реальный по умолчанию

- Форма регистрации с двумя полями: Имя и E-mail

[vospari_short_x url="http://example.com/test"]

, где атрибуты: url, width, a_aid, policy, value, practice

- Полная форма регистрации:

[vospari_full url="http://example.com/test"]

, где атрибуты: url, width, a_aid, policy, value, practice

- Виджет авторизации:

[vospari_auth url="http://example.com/test" height="300" fb="0"]

, перенаправление на любой url (обязательно), width / height - ширина/высота формы (если не указана то auto), fb - вход через facebook (0 - нет, 1 - есть), lang - язык виджета (пример: lang="/ru")


== Installation ==

1. Найти Vospari Forms плагин в WordPress директории плагинов.
2. Активировать.
3. Пользоваться шорткодами в визуальном редакторе.
4. Использовать шорткоды в php-коде шаблона, пример: -> echo do_shortcode ( "[shortcode]" ); <-
